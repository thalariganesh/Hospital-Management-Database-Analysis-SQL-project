# Hospital-Management-Database-Analysis-SQL-project
SQL project analyzing hospital patient, doctor, and admission data

## 📌 Project Overview
This project focuses on analyzing hospital data using SQL.  
The database contains patient, doctor, admission, and province information.  
The goal is to extract meaningful insights using complex SQL queries.

## 🗂 Database Tables
- Patients
- Doctors
- Admissions
- Province Names

## 🔍 Key Features
- Wrote 30+ SQL queries using JOINs, GROUP BY, HAVING, CASE, and subqueries
- Analyzed patient demographics and hospital admission trends
- Performed data cleaning by handling NULL values
- Calculated obesity using BMI logic
- Identified repeated diagnoses and admission patterns

## 🛠 Tools & Technologies
- SQL
- MySQL
- Relational Databases

## 📊 Sample Insights
- Male vs Female patient count
- Most common diagnoses
- Admission trends by date and city
- Patients admitted multiple times for the same condition
